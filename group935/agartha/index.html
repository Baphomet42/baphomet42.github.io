Console Commands


Black Ops III

Setup

  • Subscribe to a console enabler mod like this one: Console Enabler by AZA
  • On the main menu, open MODS and enable the Console Enabler mod
  • Press the backtick (`) key at any time to open the console
  • In order to allow cheats in game, run the command "/developer 2" before starting a game
    		•

    Common Commands

  • "/god" - prevents all damage
  • "/noclip" - fly through walls (hold shift to fly faster)
  • "/ui_enabled 0" and "/cg_draw2d 0" - remove HUD and crosshair
  • "/timescale 1" - speed up or slow down the game by setting a value .001 to 10
  • "/give ammo" - refill all guns and equipment
  • "/r_fog 0" - disable fog
  • "/fast_restart" - reset the level without going back to the main menu

    Weapon Codes

  • Used for the give command (Ex: "/give ray_gun") to replace your current weapon
  • Add "_upgraded" to the end of the ID for the PaP version for many guns (Ex: "/give ray_gun_upgraded")
  • Some weapons and grenades will not work properly when given through this command (such as Cymbal Monkey, Wave Gun, Q.E.D., Gersh Device, etc)
